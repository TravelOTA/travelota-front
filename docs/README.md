## Sobre el Proyecto

**TravelOTA Front** es una aplicación web B2B desarrollada con **Nuxt 4** que permite a agencias de viajes buscar, reservar y gestionar alojamientos hoteleros. La plataforma ofrece herramientas para la búsqueda avanzada de disponibilidad, proceso de checkout, gestión de reservas con estados dinámicos, y administración de agencias y usuarios.

### Stack Tecnológico

- **Framework**: Nuxt 4.3.1 (Vue 3.5, Nitro 2.13, Vite 7.3)
- **UI Library**: Nuxt UI 4.4 (componentes pre-construidos)
- **Estilos**: Tailwind CSS 4.1
- **Mapas**: Leaflet + Vue Leaflet
- **Fecha/Hora**: Vue Datepicker
- **Iconos**: Lucide, Simple Icons, Font Awesome (brands)
- **Calidad de Código**: ESLint, Prettier, Husky, lint-staged
- **Package Manager**: pnpm 10.29
